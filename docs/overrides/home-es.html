DeepDeck Ahuyama

DeepDeck es un macropad (teclado mecánico pequeño) que te permite aumentar la productividad mientras usas tus programas favortitos ya sea de trabajo, hobby, o diversión. Es totalmente OpenSource.

Con DeepDeck puedes hacer cosas como:


👩‍🎨 Contirte en un Ninja usando Photoshop/Inscape/etc/
🧑‍💻 Tener todos los comandos a la mano para tus programas de hackear (?).
👩‍🔧 Se un maestro en Freecad/Solidworks/etc/ y conviertete en un pro en 3d!
🕵 Debuguea y navega mejor en VS code para tener más productividad.

Características de DeepDeck


No queremos que DeepDeck sea como cualquier otro MacroPad, queremos que tenga SuperPoderes! 🦸 y hacerlo diferente a cualquier teclado visto antes

Estas son algunas características:
🎃 Compatibilidad con Windows, Mac, Linux, e incluso lo puedes conectar al celular.
🎃 Wireless! connecta a tu dispositivo con Bluetooth, y además tiene WiFi! 🎃 Wifi?.. Si!, Wifi! Puedes programar DeepDeck para conectarte a APIs y chequear cosas como el clima, tus subscriptores en youtube, u obtener tu lista diaria de tareas desde Notion.
🎃 No tendrás problemas de compatibilidad de software. DeepDeck no utiliza un programa especifico para ser configurado. Funciona como los modems/routers. Tiene una página web embebida donde puedes configurar todo. Lo puedes hacer desde tu computador, lo puedes hacer desde tu celular!
🎃 Estamos trabajando fuertemente para construir una gran comunidad, para que cada vez haya más personas contribuyendo al proyecto y haciendolo más versátil!

Y para el público geek estas son las características técnicas:
🤓 DeepDeck está basado en ESP32, por lo que tienen Bluetooth y WiFi integrado.
🤓 4x4 teclas mecánicas con hotswaps! que permite cambiar las teclas sin necesidad de saber soldar
🤓 2 RGB LEDs para estado de la tarjeta.
🤓 LEDs RGB en cada tecla.
🤓 2 encoders con botón incluido, y gestos como doble click y click largo.
🤓 Pantalla OLED de 128x64
🤓 Puerto I2C para integración futura de sensores.
🤓 Conector para batería LiPo.
🤓 Conector USB C
🤓 Menu de configuración integrado.
🤓 Página web integrada para configuraciones avanzadas.
