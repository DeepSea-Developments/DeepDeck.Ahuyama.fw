DeepDeck is an OpenSource MacroPad (mini mechanical keyboard) that enables you to get more productive by using it with your work/hobby/leisure programs..

Quickstart guide
In the press

Use several layers

You can program layers with different key combination to match the program you are using.

Configure it anywhere

DeepDeck has a integrated webserver to add macros, modify layers, LED status, etc. You can use any browser to program it.

Gesture sensor

Swipe your hand of top of the DeepDeck to trigger actions like magic.

OS independent

No matter if you use Windows, Linux, Mac, or even your cellphones, you can pair it and program it anywhere.

Use it with bluetooth

DeepDeck uses wifi to pair to your devices as an wireless keyboard.

Open Source

Everything on DeepDeck is open source! feel free to explore, and change it to make it your own!
